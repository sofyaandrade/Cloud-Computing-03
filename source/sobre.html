Sobre a Infraestrutura Cloud

Início Sobre o Projeto

Cenário do Trabalho

Este site representa uma plataforma de treinos online, com separação entre usuários e administradores, publicação de exercícios em ambiente público e dados protegidos em ambiente privado.

Componentes Cloud Simulados

  • VPC para separar usuários e administradores.
  • Sub-rede pública para exibição dos exercícios.
  • Sub-rede privada para banco de dados.
  • Storage para vídeos de treinos.
  • Banco relacional para usuários e programas de treino.
  • Banco não relacional para sugestões e notificações personalizadas.
  • VM de exibição dos treinos e VM de análise de desempenho.
